GrobPaint was started because Paint.NET is not available on macOS.

**Why:** The user wanted a Paint.NET-like image editor that works on macOS (and other platforms).
**How to apply:** When writing docs or making design decisions, frame the project as a cross-platform alternative to Paint.NET. The macOS gap is the core motivation.
